It is not described in the kikuchipy documentation how one can navigate in RGBA images. The description can be added to the "Any image" part in the tutorial "Visualizing patterns" (https://kikuchipy.org/en/stable/tutorials/visualizing_patterns.html#Any-image)

The image quality map can be used as overlay on the RGB image to visualize the orientation of the grains and their morphology in the same image

>>> maps_iq = iq.ravel() # Flat array required by orix
>>>
>>> maps_rgb_z = rgb_z.reshape(-1, 3)
>>>
>>> fig = s.xmap.plot(maps_rgb_z, overlay=maps_iq, return_figure=True)
>>> maps_rgb_z_iq = fig.axes[0].images[0].get_array()
>>> 
>>> nav_rgb_z_iq = kp.draw.get_rgb_navigator(maps_rgb_z_iq)
>>>
>>> s.plot(nav_rgb_z_iq)
